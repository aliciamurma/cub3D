# cub3D


# Instruccions
Compile code mandatory version
```shell
make
```
Compile code bonus version
```shell
make bonus
```
Ejecute cub3D mandatory version
```shell
make run
```
Ejecute cub3D bonus version
```shell
make run_b
```
